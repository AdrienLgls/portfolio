Java - Classification de dépêches

L'objectif de ce projet était d'implémenter un programme de classification automatique de dépêches, aussi fiable et rapide que possible.

Nous avons travaillé sur le développement du programme en réalisant plusieurs classes, chacune ayant un rôle spécifique :
Génération automatique d'un fichier texte : Cette classe est responsable de la création automatique d'un fichier texte contenant le classement des dépêches.
Réalisation du classement à partir d'une base de données de test : Cette classe utilise une base de données de test pour effectuer le classement initial des dépêches, permettant de vérifier et ajuster les algorithmes de classification.

Exécution sur une base de données différente pour le classement final : Cette classe prend en charge l'exécution du programme de classification sur une base de données différente de celle utilisée pour les tests, pour valider la robustesse et la généralisation de notre solution.

Nous avons également optimisé notre programme afin de réduire le temps d'exécution, en améliorant les algorithmes de traitement et en optimisant les structures de données utilisées. Cela a permis de rendre notre programme non seulement plus rapide, mais également plus efficace en termes de consommation de ressources.
